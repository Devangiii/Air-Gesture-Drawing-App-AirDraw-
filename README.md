# Air-Gesture-Drawing-App-AirDraw-
AirDraw opens your camera, tracks your hand with MediaPipe, and maps gestures to actions: draw with one finger, move with two, pause with a fist, and erase with a pinch. A floating toolbar lets you pick color, brush size, opacity, clear the canvas, or save artwork, while live status, HUD cues, and webcam preview keep control smooth and intuitive UX
